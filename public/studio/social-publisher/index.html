      Connection walkthrough

      One-time setup

      Nothing is posted during setup. Start with Facebook and Instagram, then add the other platforms only when Josh is ready.

      1. Confirm the exact profile or business page to connect, and make sure Josh is an owner or administrator.
      2. Use the account's normal login in the production app to approve only the requested publishing permissions.
      3. Return here and confirm the account name shown by the connection matches the intended destination.
      4. Send one test post to a non-public or test destination before enabling real publishing.
